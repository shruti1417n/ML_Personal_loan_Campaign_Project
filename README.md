# ML_Personal_loan_Campaign_Project

Course: Machine Learning
This project leverages data analytics to optimize AllLife Bank's marketing strategy. We aim to predict liability customers' likelihood of purchasing personal loans and identify key customer attributes influencing loan adoption. By targeting the right customer segments, we can provide actionable insights to increase loan conversions and drive revenue growth.
